# vuetest

> A Vue.js project

## Build Setup

``` bash
#install node.js
brew install node

#go app folder
#install dependencies
npm install

#json server run
npm run server

# serve with hot reload at localhost:8080
npm run dev

```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
